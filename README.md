# GroSphere: AI-Powered Urban Gardening "Mission Control"

**GroSphere** is a high-tech ecosystem designed for urban gardeners living in small spaces. It combines cinematic UI aesthetics with Cloud AI, AR Guidance, and IoT Telemetry to create a "Mission Control" experience for plant care.

---

## 🚀 Key Features
- **Mission Control Dashboard:** Real-time telemetry visualization for your urban ecosystem.
- **Cloud AI Diagnostics:** Plant identification and disease diagnosis powered by Hugging Face.
- **AR Smart Guidance:** Holographic overlays for precise pruning and spatial measurements.
- **IoT Integration:** Real-time monitoring of soil moisture, light, and temperature.
- **Community Marketplace:** A P2P seed swap and marketplace for urban gardening essentials.
- **Gardening Copilot:** An AI-powered conversational assistant that knows your garden and local weather.

## 🛠️ Technical Stack
- **Monorepo:** Turborepo
- **Mobile:** React Native (Expo), Reanimated, WatermelonDB (Offline-first).
- **Web:** Next.js (App Router), Framer Motion, Tailwind CSS.
- **Backend:** Supabase (Auth, Postgres, Realtime, Edge Functions).
- **AI/ML:** Hugging Face Inference API.
- **Payments:** Stripe API.

## 📦 Project Structure
- `apps/mobile`: Expo mobile application.
- `apps/web`: Next.js web application.
- `packages/ui`: Shared React/React Native components.
- `packages/shared`: Shared types and business logic.
- `supabase`: Database migrations and Edge Functions.
